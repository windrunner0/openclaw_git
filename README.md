## 📊 A股量化监控系统

本项目集成了**A股机构级量化交易监控系统 Ultimate V5**，自动生成实时量化监控报告。

### 最新报告

**[查看最新量化监控报告 →](quant-monitor-report-20260311_010124.html)**

### 报告内容

✅ **市场情绪分析**
- 市场周期判断（强势期/弱势期等）
- 情绪等级评分（1-5级）
- 风险偏好评估
- 涨跌停比例统计

✅ **数据监控**
- 涨停池实时监控
- 跌停池实时监控
- 连板股识别（≥2连板）
- 龙虎榜数据追踪
- 行业资金流向分析

✅ **可视化图表**
- 行业资金Top10柱状图
- 市场情绪饼图
- 龙头股散点图

✅ **Excel详细报告**
- 包含全部因子数据
- 排名榜单
- 趋势分析
- 可直接下载分析

### 系统特性

- 🔄 **自动更新**: 定时任务自动执行（早盘、午盘、收盘）
- 📈 **实时数据**: 基于AKShare获取最新市场数据
- 🎯 **多因子分析**: 龙头股评分、资金流向、情绪判断
- 💾 **历史数据**: SQLite数据库存储历史记录
- 📱 **响应式设计**: 支持PC和移动端查看
